Portfolio

Here is a list of selected projects.

EEG-based P300 Truth Elicitor

Devpost | GitHub Repo
This was a project completed with Sandra Karon, Darren Vawter, Steven Veld, Ben Falkenburg, and Davis Sandberg that won 1st place in natHACKS: Canada's largest neurotechnology hackathon in 2023. This is a brain-computer interface (BCI) game designed to uncover hidden information from a subject. This project made use of a personally-designed bootstrap signal correlation algorithm.

EMG Decomposition

GitHub Repo
This was a project completed for my Neuroengineering class (BIOENGR M260 @ UCLA). In this project, I had to decompose electromyography (EMG) signals into individual motor unit action potentials (MUAPs). I used basic signal processing, dimension reduction, and clustering techniques in Python to do so.

Exploring Differential Effects of Gamma Stimulation on Alzheimer's Disease

InT BioHackathon | GitHub Repo
In this project completed with John Walkiewicz and Logan Umaguing, we designed a research proposal in which we aimed to evaluate the efficacy of multimodal gamma stimulation on Alzheimer's Disease (AD). With this project, we achieved runner up at the InTranscription BioHackathon 2023.

Vehicular Remote Online Ocular brain-Machine interface (VROOM)

GitHub Repo
This is the CruX Neurotech project I did with Avantika Aggarwal, Hanna Boughanem, Paige Lee, Sree Nagaraj, Srivarsha Rayasam, Anish Thalamati, and Audrey Ngai. Here, we sought to control a remote-control car with EEG signals. We presented our work at the 2023 California Neurotechnology Conference.

GIS Coursework

GitHub Repo
This is a collection of my work in Introductory GIS (GEOG 7 @ UCLA). This was all done in QGIS and Adobe Illustrator.

Statistical Modeling Projects

GitHub Repo
These are statistical machine learning projects done with Andra Velea and Aryan Mistry in Statistical Models and Data Mining (STATS 101C @ UCLA). We competed in class-wide regression and classification task Kaggle competitions and received first place for testing accuracy.
